## Math

| 方法          | 描述                                       |
| ------------- | ------------------------------------------ |
| Math.round()  | 四舍五入为最接近整数                       |
| Math.pow(x,y) | x的y次幂                                   |
| Math.sqrt()   | x的平方根                                  |
| Math.abs()    | x的绝对(正)值                              |
| Math.celi()   | x上舍入最接近的整数                        |
| Math.floor()  | x下舍入最接近的整数                        |
| Math.sin()    | 角的正弦                                   |
| Math.cos()    | 角的余弦                                   |
| Math.min()    | 查找参数列表中的最低值                     |
| Math.max()    | 查找参数列表中的最高值                     |
| Math.random() | 返回介于0（包括）与1（不包括）之间的随机数 |
